EmailPwned API Documentation

Base URL: https://emailpwned-api.vercel.app/v1/

1. Check Breaches by Email (Email Shield)

Endpoint

/email-shield/

Method

GET

Description

This endpoint checks for breaches associated with the provided email address.

Query Parameters

  • email (required): The email address to check for breaches.

Example Request

GET /email-shield/?email=aryanstha4859%40gmail.com

Response

{
  "Breaches_Count": 2,
  "Breaches_Records": [
    {
      "Title": "Breach Example 1",
      "Domain": "example.com",
      "BreachDate": "2022-06-01"
    },
    {
      "Title": "Breach Example 2",
      "Domain": "example2.com",
      "BreachDate": "2023-03-05"
    }
  ]
}

4. Metrics

Endpoint

/metrics/

Method

GET

Description

Returns metrics data about all breaches, including a count of breached records and affected domains.

Example Request

GET /metrics/

Response

{
  "Total_Breaches": 10500,
  "Affected_Domains": 8500
}

8. Email Domain Verification

Endpoint

/emailVerification/

Method

GET

Description

Verifies domain ownership for a given email address's domain.

Query Parameters

Response

{
  "Verification_Status": "Success"
}

9. Slack Configuration

Endpoint

/slack-config/

Method

GET

Description

Sets up Slack integration for breach alerts.

Query Parameters

  • email (required): The email address to associate with the Slack configuration.
  • domain (required): The domain to associate with the Slack configuration.
  • token (required): The authentication token for Slack API.

Example Request

GET /slack-config?email=example@gmail.com&domain=example.com&token=abcd1234

Response

{
  "Slack_Integration": "Configured Successfully"
}
